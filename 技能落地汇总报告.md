## 技能摘要

### 1️⃣ ssh-remote (SSH 远程服务器管理)

**功能**:
- SSH 连接管理（多主机配置）
- 远程命令执行（安全沙箱）
- 文件传输（scp/sftp）
- 服务监控

**安全属性**:
- 🔴 红线：删除/格式化、重启生产服务器 → MAX 明确确认
- 🟡 黄线：执行未知命令、访问敏感目录 → 先问询
- 🟢 绿线：查看日志、状态检查 → 直接执行

**配置示例** (ssh_hosts.yaml):
```yaml
hosts:
  - name: home-server
    host: 192.168.1.100
    user: admin
    safety_level: green
    
  - name: vps-prod
    host: x.x.x.x
    user: root
    safety_level: yellow

green_commands:
  - "df -h"
  - "free -m"
  - "top -bn1"
  - "ps aux"
  - "systemctl status *"

blacklist_commands:
  - "rm -rf /"
  - "mkfs.*"
  - "reboot"
```

### 2️⃣ auto-backup (自动备份与同步)

**功能**:
- 定时备份 (cron 配置)
- 飞书文档同步
- 增量备份 (节省空间)
- 恢复验证

**备份任务**:
| 任务 | 源 | 频率 | 安全级别 |
|------|-----|------|----------|
| workspace-daily | 工作区 | 每天 02:00 | 🟢 |
| workspace-weekly | 工作区 | 每周日 03:00 | 🟢 |
| feishu-docs | 飞书文档 | 每天 03:00 | 🟡 |
| config-files | 系统配置 | 每天 04:00 | 🟡 |

**配置示例** (backup_config.yaml):
```yaml
backups:
  - name: workspace-daily
    source: ~/.openclaw/workspace/
    destination: ~/backups/workspace/daily/
    schedule: "0 2 * * *"
    type: incremental
    retention:
      days: 30
      versions: 7
    exclude:
      - "*.log"
      - "node_modules/"
      - ".git/"
```

## 安全设计

### 统一安全级别

| 级别 | 定义 | 执行方式 |
|------|------|----------|
| 🔴 红线 | 删除、覆盖、对外发布、重启生产 | MAX 明确确认，无确认不执行 |
| 🟡 黄线 | 隐私数据、敏感目录、外部同步 | 先问询 MAX，确认后执行 |
| 🟢 绿线 | 读取、统计、内部报告、状态检查 | 直接执行，事后简报 |
