- 安全漏洞扫描

- 最佳实践建议

### 2. 调试辅助
- 错误日志分析
- 问题定位
- 修复建议
- 调试脚本生成

### 3. 代码生成

- 模板代码
- API 客户端

- 代码优化建议

- 测试用例生成

- Mock 数据生成
