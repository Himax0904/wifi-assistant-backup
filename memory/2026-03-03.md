- MAX 提出：NotebookLM 独立知识库 + 多 AI 灵活调用
- 需求：知识库与 AI 模型解耦，便于迁移
- 决策：采用混合架构（NotebookLM + 本地 + Git 备份）

### 架构设计

```
应用层（各角色）
    │
    ├─→ NotebookLM（主动分析）
    │   • AI 分析
    │   • 结构化输出

    └─→ 本地知识库（归档存储）
```
